# Python Array Algorithms

A collection of Python programs demonstrating core array manipulation techniques.

## Programs

### `duplicates.py`
Takes a list of integers as input and returns the same list with all duplicate values removed, preserving the original order.

### `missing_number.py`
Accepts a sequence of integers with one value missing and identifies the missing number using arithmetic sum comparison.

### `rearranged.py`
Rearranges a sorted array so that the largest element is at index 0, the smallest at index 1, the second largest at index 2, and so on — alternating high/low.

### `gridio.py`
Reads a 9×9 grid of integers from user input and allows repeated coordinate queries, returning the value at each specified (x, y) position.

## How to Run

```bash
python duplicates.py
python missing_number.py
python rearranged.py
python gridio.py
```
